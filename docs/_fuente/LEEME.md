# Fuente de la web de Imago

- `index.html` (en la carpeta de arriba) es la web FINAL: un único archivo
  autónomo (imágenes y fuente incrustadas). Súbelo tal cual a cualquier hosting.
- Para EDITAR: cambia `imago_site.template.html` y ejecuta `python build.py`,
  que regenera `../index.html`.
- `assets/` tiene las imágenes ya optimizadas (WebP) y la fuente (Cantarell woff2).
- Las capturas originales sin optimizar están en `../imagenes_web/`.
